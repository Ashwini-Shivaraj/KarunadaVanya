# KarunadaVanya

KarunadaVanya is an Android app. It is made using Kotlin and Jetpack Compose. The app helps people learn about wildlife in Karnataka and stay safe near forests.

## Features

* Wildlife Wiki

Learn about animals in Karnataka forests. You can find information on animals.

* Movement Alert System

If you see an animal you can report it with its location. This helps others stay safe.

* Firebase Realtime Database

The app uses Firebase to show alerts in time. This way people can see alerts as they happen.

* Alert History

You can view alerts that were reported before. This helps you see what kinds of animals are in the area.

* Safety Guide

The app gives you safety tips for animals. This helps you stay safe if you encounter an animal.

* Forest Sounds

You can listen to sounds of nature and wildlife.

* Karnataka Forest Map

The app has a map of Karnataka forests. You can use it to find wildlife areas.

## Technologies Used

* Kotlin

* Jetpack Compose

* Firebase Realtime Database

* Android Studio

## Objective

The main goal of KarunadaVanya is to spread awareness, about wildlife. We want to help people stay safe and informed near forests. KarunadaVanya aims to make people aware of wildlife conservation.
